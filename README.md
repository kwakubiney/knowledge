# Knowledge

Personal repo for working with my knowledge vault.

## What This Is

- This repo holds scripts and OpenCode skills.
- The actual vault lives at `~/Documents/knowledge-vault`.

## Main Skills

- `capture-idea` - capture project ideas quickly and turn them into actionable notes
- `process-book` - route book work to the right workflow
- `process-book-chapters` - turn pasted chapters into chapter notes
- `process-book-highlights` - process Apple Books highlights into one book highlights note
- `process-youtube` - turn YouTube videos into clean vault notes
- `vault-context` - load only the context needed for a task
- `vault-write` - keep note formatting and structure consistent
- `update-context` - record new understanding or changes in a domain
- `synthesize-knowledge` - create higher-level synthesis across notes

## Main Paths

- Repo: `~/Desktop/knowledge`
- Vault: `~/Documents/knowledge-vault`
